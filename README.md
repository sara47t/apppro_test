## gitでブランチを作る

ブランチとは枝のこと、開発において違う系統で行いたい場合はブランチを作ります。
かっこよく言うとブランチを切るといいます。

- 今のブランチを見る
'''
git branch
'''

- ブランチを作成する
'''
git branch test_branch
'''

- ブランチを移動
'''
git checkout test_branch
'''
